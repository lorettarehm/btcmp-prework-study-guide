HTML

  • The head element contains information about the webpage.
  • The meta tag defines metadata about an HTML document. It is data (information) about data. It will not be displayed on the page, but is machine parsable.
  • The body element represents the visible content shown to the user.
  • The link tag will connect the html file to the css and js files.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The width property sets the width of an element. It does not include padding, borders, or margins.
  • The height property sets the height of an element. It does not include padding, borders, or margins.
  • The background-color property sets the background color of an element. It is the total size of the element, including padding and border (but not the margin).
  • The color property specifies the color of text.
  • The text-align property specifies the horizontal alignment of text in an element.
  • The display property specifies the display behavior (the type of rendering box) of an element.
  • The margin-left property sets the left margin of an element.
  • The margin-right property sets the right margin of an element.
  • The padding-left property sets the left padding (space) of an element.
  • The font-size property sets the size of a font.
  • The margin property sets the margins for an element, and is a shorthand property for: margin-top, margin-right, margin-bottom and margin-left.
  • The border property is a shorthand property for: border-width, border-style (required) and border-color. If border-color is omitted, the color applied will be the color of the text.
  • The box-shadow property attaches one or more shadows to an element.

Git

  • git status: checks what branch we are currently on
  • git checkout main: goes to main branch
  • git checkout -b feature/branchName: creates the branch from main
  • git add -A: commit all files in the working branch
  • git commit -m "Commit comment": will commit your changes and provide the comment as description for the commit.
  • git pull origin main: pull in the base branch before pushing changes
  • git checkout -b branch-name: creates a new branch and switches to it
  • git push origin feature/branchName: push up changes done on branch

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Conditional statements and loops allow us to control the order in which code is executed.
  • A value is considered falsy if it is one of the following values: 0, -0, On, "", null, undefined, NaN, or a Boolean data type with the value of false. All other values, including a Boolean with the value of true, are considered truthy.
